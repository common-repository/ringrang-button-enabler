== Description ==
This plugin allows for the use of one or more RingRang buttons / icons on any WordPress site. The RR button when placed on a WP site allows customers and friends alike to send a notification immediately to the button/icon owner via email, text message or stored in a database. The notifications contains the phone / text number and an optional 120 characters.

== Installation ==
1. Upload `RingRang.php` and `button.js` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. When you want to use a button, add the following to the HTML section of your post (notice XXXXX should be replaced with your button ID):
`<script type="text/javascript">RingRang.button("XXXXX");</script>`
